# Projects Directory — Schema

每個專案一個 Markdown 檔案，格式如下：

```
projects/{slug}.md
```

## Frontmatter 格式

```yaml
---
name: 專案名稱
slug: project-slug
status: active    # active | paused | done | archived
started: YYYY-MM-DD
owner: Clawton    # agent 或 human 名稱
tags: [research, coding, automation]
description: 一行簡短描述
---
```

## Body 格式（Markdown）

### 目標
說明這個專案要達成什麼。

### 進度
- [x] 已完成事項
- [ ] 待辦事項

### 相關資源
- 連結、檔案、備註
